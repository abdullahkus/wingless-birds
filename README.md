# Wingless Birds
This is a game where horses compete in 8 different lanes.

[Check out the live example.](https://vue-bootcamp-final-project-1x9nbh7c8-abdullahkus.vercel.app/)

### General Features

- Custom horse names
- Live leaderboard
- Results list with lane number and horse name

### Technologies

- Vue.js
- Pinia
- Scss
